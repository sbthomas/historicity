# SBL Greek New Testament (SBLGNT)

The files in this directory contain the Greek text of the seven undisputed
Pauline letters (Romans, 1 Corinthians, 2 Corinthians, Galatians,
Philippians, 1 Thessalonians, Philemon), extracted from the **SBL Greek
New Testament** (SBLGNT), edited by Michael W. Holmes.

Files were downloaded from `raw.githubusercontent.com/LogosBible/SBLGNT/master/data/sblgnt/text/`
on 2026-04-17.  No alterations have been made to the Greek text itself;
the `build_supplement.py` script strips editorial sigla (apparatus
variants) at rendering time for readability but does not modify the
underlying files in this directory.
